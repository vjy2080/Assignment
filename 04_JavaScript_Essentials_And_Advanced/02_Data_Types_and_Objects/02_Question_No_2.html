Question No.2
Is array copied?
let fruits = ["Apples", "Pear", "Orange"];
// push a new value into the "copy" let shoppingCart = fruits;
shoppingCart.push("Banana");
// what's in fruits? alert( fruits.length ); // ?

ANSWER

Yes, Array is copied as a new variable "shoppingCart"

shoppingCart => ['Apples', 'Pear', 'Orange', 'Banana']

shoppingCart.length => "4"

fruits => ['Apples', 'Pear', 'Orange', 'Banana']

fruits.length => "4"

alert( fruits.length ); is shown on startup
