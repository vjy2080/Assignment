Question No.2
What is promises

ANSWER

a promise is an object that represents the eventual outcome of an asynchronous task. It can be in one of three states:

  1. Pending: The initial state when the asynchronous task hasn't completed yet.
  2. Fulfilled: The state when the asynchronous task has completed successfully, and the promise has a resulting value.
  3. Rejected: The state when the asynchronous task has encountered an error or failure, and the promise has a reason for the failure.
